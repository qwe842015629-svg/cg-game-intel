## 🔧 常见操作

### 添加新轮播图

1. 进入后台: `/admin/main/banner/`
2. 点击"添加轮播图"
3. 填写必填字段:
   - 标题
   - 图片
4. 配置可选字段:
   - 描述
   - 徽章
   - 按钮文字和链接
5. 设置排序（数字越小越靠前）
6. 保存

### 修改轮播图顺序

**方法1**: 直接修改排序值
- 在列表页直接修改"排序"列

**方法2**: 拖拽排序
- 在列表页拖拽行来重新排序

### 禁用轮播图

1. 进入编辑页面
2. 将"状态"改为"禁用"
3. 保存

禁用的轮播图不会在前端API中返回。

### 设置默认轮播图

1. 进入编辑页面
2. 勾选"是否默认轮播图"
3. 保存

系统会自动取消其他轮播图的默认状态。

## ⚠️ 注意事项

### 1. 图片尺寸
- **推荐**: 1920x600px
- **格式**: JPG, PNG, WebP
- **大小**: < 500KB（优化加载速度）

### 2. 默认轮播图
- 只能有一个默认轮播图
- 设置新的默认时，系统会自动取消旧的

### 3. 排序规则
- 数字越小越靠前
- 相同排序值按创建时间倒序

### 4. 状态管理
- 只有"启用"状态的轮播图会在API中返回
- 前端不会显示"禁用"的轮播图

## 🐛 故障排除

### 问题1: 前端不显示轮播图

**检查**:
1. Django服务器是否运行: http://127.0.0.1:8000/
2. API是否返回数据: http://127.0.0.1:8000/api/banners/
3. 轮播图状态是否为"启用"
4. 浏览器控制台是否有错误

### 问题2: 图片不显示

**检查**:
1. 图片是否已上传
2. MEDIA_URL配置是否正确
3. 图片文件是否存在于media/banners/目录

### 问题3: 后台无法访问

**检查**:
1. 是否已登录管理员账户
2. URL是否正确: /admin/main/banner/
3. 菜单中是否显示"轮播图管理"
