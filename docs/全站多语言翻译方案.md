### 步骤 3：修改组件使用翻译函数

#### 3.1 Layout.vue 修改

**修改前**（硬编码中文）：
```vue
<RouterLink to="/profile">
  个人中心
</RouterLink>
<button @click="handleLogout">
  退出登录
</button>
```

**修改后**（使用翻译）：
```vue
<RouterLink to="/profile">
  {{ $t('userProfile') }}
</RouterLink>
<button @click="handleLogout">
  {{ $t('logout') }}
</button>
```

#### 3.2 AuthDialog.vue 修改

**修改前**：
```vue
<Input v-model="username" placeholder="请输入用户名" />
<Button>立即登录</Button>
```

**修改后**：
```vue
<Input v-model="username" :placeholder="$t('pleaseEnterUsername')" />
<Button>{{ $t('loginNow') }}</Button>
```

#### 3.3 使用 Composition API（推荐）

```vue
<script setup>
import { useI18n } from '@/composables/useI18n'

const { t } = useI18n()
</script>

<template>
  <div>
    <h1>{{ t('userProfile') }}</h1>
    <button @click="logout">{{ t('logout') }}</button>
  </div>
</template>
```

## 🔧 常见问题解决

### 问题 1：翻译脚本运行失败

**原因**：API 密钥过期或网络问题

**解决**：
```bash
# 检查 API 状态
python scripts/test_translation_api.py

# 检查配置文件
cat .env | grep TRANSLATION
```

### 问题 2：翻译质量不佳

**原因**：机器翻译的局限性

**解决**：
- 手动修正关键术语
- 使用专业翻译服务
- 咨询母语者

### 问题 3：部分文本未翻译

**原因**：组件中硬编码中文未替换

**解决**：
```bash
# 搜索所有 Vue 文件中的硬编码中文
grep -r "[\u4e00-\u9fa5]" frontend/src/**/*.vue
```

### 问题 4：翻译键冲突

**原因**：新键与已有键重名

**解决**：
- 使用更具描述性的键名
- 添加组件前缀，如 `layout_userProfile`
