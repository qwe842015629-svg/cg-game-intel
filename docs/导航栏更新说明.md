# 导航栏下拉菜单更新说明

## ✅ 完成的功能

### 1. 文字字体放大
- 导航栏文字从 `text-sm` 提升至 **`text-base`**
- 移动端菜单文字也同步提升至 `text-base`
- 更醒目，更易点击

### 2. 游戏下拉菜单
**桌面端：**
- 鼠标悬停在"游戏"（充值）上时自动显示下拉菜单
- 包含三个选项：
  - **国际服** - 跳转到 `/recharge?region=international`
  - **港台服** - 跳转到 `/recharge?region=hktw`
  - **东南亚服** - 跳转到 `/recharge?region=sea`

**移动端：**
- 点击"游戏"展开下拉选项
- 带箭头旋转动画
- 点击子项后自动关闭菜单

### 3. 资讯下拉菜单
**桌面端：**
- 鼠标悬停在"资讯"上时自动显示下拉菜单
- 包含四个文章分类：
  - **游戏资讯** - `/news?category=游戏资讯`
  - **攻略教程** - `/news?category=攻略教程`
  - **充值指南** - `/news?category=充值指南`
  - **活动公告** - `/news?category=活动公告`

**移动端：**
- 点击"资讯"展开下拉选项
- 带箭头旋转动画
- 点击子项后自动关闭菜单

## 🔧 技术实现

### 响应式状态管理
```javascript
const showGamesMenu = ref(false)      // 桌面端游戏菜单
const showNewsMenu = ref(false)       // 桌面端资讯菜单
const mobileGamesOpen = ref(false)    // 移动端游戏菜单
const mobileNewsOpen = ref(false)     // 移动端资讯菜单
```

### 鼠标事件（桌面端）
```html
<div @mouseenter="showGamesMenu = true" @mouseleave="showGamesMenu = false">
  <!-- 菜单内容 -->
</div>
```

### 点击切换（移动端）
```html
<button @click="mobileGamesOpen = !mobileGamesOpen">
  <!-- 带旋转箭头 -->
  <svg :class="{ 'rotate-180': mobileGamesOpen }">
</button>
```

## 📝 后续可扩展

### 1. 充值页面筛选逻辑
在 `RechargePage.vue` 中添加：
```javascript
const route = useRoute()
const selectedRegion = computed(() => route.query.region)

// 根据 region 筛选游戏列表
const filteredGames = computed(() => {
  if (!selectedRegion.value) return games
  return games.filter(game => game.region === selectedRegion.value)
})
```

### 2. 资讯页面筛选逻辑
在 `NewsPage.vue` 中添加：
```javascript
const route = useRoute()
const selectedCategory = computed(() => route.query.category)

// 根据 category 筛选文章
const filteredArticles = computed(() => {
  if (!selectedCategory.value) return articles
  return articles.filter(article => article.category === selectedCategory.value)
})
```

### 3. 动态获取分类
可以从后端 API 获取分类数据：
```javascript
// 获取游戏区服
const gameRegions = await fetch('/api/game-regions/')

// 获取文章分类
const articleCategories = await fetch('/api/articles/categories/')
```
