| 尺寸 | 1920x600px | 宽高比 3.2:1 |

   - 下载并调整尺寸为1920x600px

3. **使用图片编辑软件**
   - Photoshop
   - GIMP（免费）
   - 在线工具：https://www.iloveimg.com/resize-image

## 📝 字段说明

### 基本信息

#### 标题 (必填)
- **用途**: 轮播图主标题
- **示例**: "和平精英点券"
- **显示位置**: 轮播图左侧，大字体

#### 描述 (可选)
- **用途**: 轮播图副标题/说明文字
- **示例**: "国民级射击手游，热卖中，充值送稀有皮肤"
- **显示位置**: 标题下方，小字体
- **长度**: 建议50-100字

#### 徽章 (可选)
- **用途**: 吸引眼球的标签
- **示例**: "热卖中"、"新品上市"、"热门畅销"、"特别优惠"
- **显示位置**: 标题上方，彩色小标签
- **颜色对应**:
  - 热卖中 → 粉红色
  - 新品上市 → 青色
  - 热门畅销 → 橙色
  - 特别优惠 → 绿色

#### 图片 (必填)
- **用途**: 轮播图背景图
- **要求**: 1920x600px，< 500KB
- **效果**: 覆盖整个轮播图区域

### 按钮配置

#### 主按钮
- **文字**: 如"立即抢购"、"立即充值"
- **链接**: 点击后跳转的页面，如"/recharge"
- **样式**: 青色渐变按钮

#### 次按钮  
- **文字**: 如"浏览更多"、"查看详情"
- **链接**: 点击后跳转的页面，如"/games/2"
- **样式**: 粉红色边框按钮

### 显示设置

#### 状态
- **启用**: 在前端API和页面显示
- **禁用**: 不显示，但保留数据

#### 排序
- **规则**: 数字越小越靠前
- **示例**: 1排在2前面

#### 默认轮播图
- **作用**: 标记为首选轮播图
- **限制**: 只能有一个默认轮播图

## 📞 常见问题

### Q1: 上传图片后看不到？
**A**: 检查：
1. 文件格式是否支持（JPG/PNG/WebP）
2. 文件大小是否合理（< 10MB）
3. media/banners目录是否有写入权限

### Q2: 图片显示但前端看不到？
**A**: 检查：
1. 轮播图状态是否为"启用"
2. 浏览器是否缓存了旧数据（F5刷新）
3. API是否返回了图片URL

### Q3: 想要不同尺寸的图片？
**A**: 
- 模型支持任意尺寸
- 但推荐1920x600px以保持一致性
- 前端会自动适配不同尺寸
