## 🔧 技术实现

### 文件结构
```
frontend/
├── src/
│   ├── i18n/
│   │   └── locales.ts          # 所有翻译内容
│   ├── plugins/
│   │   └── i18n.ts             # i18n 插件配置
│   ├── stores/
│   │   └── language.ts         # 语言状态管理
│   └── views/                  # 各页面组件
│
scripts/
├── translation_cache/          # 翻译缓存（9个JSON文件）
│   ├── en.json
│   ├── ja.json
│   ├── ko.json
│   ├── th.json
│   ├── vi.json
│   ├── zh-TW.json
│   ├── fr.json
│   └── de.json
└── add_games_page_translations.py  # 翻译脚本
```

### 使用方式

在 Vue 组件中：
```vue
<template>
  <div>
    <h1>{{ $t('gameRecharge') }}</h1>
    <p>{{ $t('selectGameToRecharge') }}</p>
    <button>{{ $t('startRecharge') }}</button>
  </div>
</template>
```

切换语言：
```typescript
import { useLanguageStore } from '@/stores/language'

const languageStore = useLanguageStore()
languageStore.setLocale('ja')  // 切换到日语
```

## 📝 维护说明

### 添加新翻译
1. 在 `locales.ts` 的 `zh-CN` 部分添加新键
2. 运行 `python scripts/add_games_page_translations.py` 自动翻译到其他语言
3. 或手动在每个语言块中添加对应翻译

### 修改现有翻译
直接编辑 `frontend/src/i18n/locales.ts` 文件中对应语言的翻译即可

### 翻译质量优化
如果AI翻译不够准确，建议：
1. 找专业翻译人员校对
2. 请母语用户测试反馈
3. 根据用户反馈调整翻译
