5. **测试所有 9 种语言**

### 3. 验证混合翻译策略

在浏览器控制台（F12）运行以下测试：

```javascript
// 1. 获取当前语言
console.log('当前语言:', app.$languageStore?.currentLocale)

// 2. 测试自定义翻译（应该使用 locales.ts）
console.log('网站名称:', app.$t('siteName'))

// 3. 测试轮播图翻译（应该使用 locales.ts）
console.log('轮播图标题1:', app.$t('carouselTitle1'))

// 4. 切换语言
if (app.$languageStore) {
  app.$languageStore.setLocale('en')
  console.log('切换到英语:', app.$t('siteName'))
  
  app.$languageStore.setLocale('ja')
  console.log('切换到日语:', app.$t('siteName'))
}
```

## 🔍 问题排查

### 如果某些文本没有翻译

1. **检查浏览器控制台**
   - 是否有错误信息？
   - 检查网络请求是否正常

2. **检查翻译键**
   ```javascript
   // 在控制台运行
   console.log(app.$t('problemKey'))
   ```

3. **验证语言是否切换**
   ```javascript
   // 在控制台运行
   console.log('Current locale:', app.$languageStore?.currentLocale)
   ```

### 如果页面无法加载

1. 检查开发服务器是否运行
2. 检查端口是否正确（http://localhost:5188）
3. 检查浏览器控制台错误

## 🚀 下一步

测试通过后，您可以：

1. **添加更多翻译**

   - 添加新的翻译键值对

2. **扩展到其他页面**
   - 使用 `import { useI18n } from '@/composables/useI18n'`
