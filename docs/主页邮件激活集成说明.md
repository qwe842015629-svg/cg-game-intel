## 🚀 使用流程

### 1. 用户在主页点击登录按钮

任何页面的"Login"按钮都会打开 [`AuthDialog`](e:\小程序开发\游戏充值网站\frontend\src\components\AuthDialog.vue) 弹窗。

### 2. 切换到注册标签

在弹窗中点击"注册"标签。

### 3. 填写注册信息

- **用户名**：输入昵称
- **邮箱**：输入真实邮箱（用于接收激活邮件）
- **密码**：至少8位
- **确认密码**：再次输入密码

### 4. 点击注册按钮

系统会：
- 验证密码是否匹配
- 验证密码长度
- 调用 Djoser API 创建账号
- 自动发送激活邮件

### 5. 查看待激活提示

注册成功后，弹窗自动切换为**待激活提示界面**，显示：
- ✅ 注册成功动画
- 📧 邮件发送确认
- 📋 详细的操作步骤指南
- ⚠️ 注意事项提示
- 🔘 快捷操作按钮

### 6. 打开邮箱激活

点击"📧 打开邮箱"按钮：
- 自动识别邮箱类型
- 在新标签页打开对应邮箱网站
- 用户查找激活邮件
- 点击邮件中的激活按钮

### 7. 自动激活并跳转

点击激活链接后：
- 自动跳转到 [`ActivateAccountPage`](e:\小程序开发\游戏充值网站\frontend\src\views\ActivateAccountPage.vue)
- 显示"正在激活..."
- 激活成功显示 🎉 提示
- 5秒倒计时后自动跳转登录页
- 或点击"立即登录"按钮手动跳转

### 8. 登录系统

- 在主页再次点击"Login"按钮
- 输入邮箱和密码
- 登录成功后关闭弹窗
- 用户状态更新，显示用户头像

## 🔧 API 端点使用

### 注册 API
```typescript
POST http://localhost:8000/api/auth/users/
{
  "username": "testuser",
  "email": "test@example.com",
  "password": "password123",
  "re_password": "password123"
}
```

### 登录 API
```typescript
POST http://localhost:8000/api/auth/token/login/
{
  "email": "test@example.com",
  "password": "password123"
}

Response:
{
  "auth_token": "9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b"
}
```

### 获取用户信息 API
```typescript
GET http://localhost:8000/api/auth/users/me/
Headers:
  Authorization: Token 9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b

Response:
{
  "id": 1,
  "username": "testuser",
  "email": "test@example.com"
}
```

### 激活账号 API
```typescript
POST http://localhost:8000/api/auth/users/activation/
{
  "uid": "MQ",
  "token": "c4u-39b9d1fc0e4e9f0ba870"
}
```

### 退出登录 API
```typescript
POST http://localhost:8000/api/auth/token/logout/
Headers:
  Authorization: Token 9944b09199c62bcf9418ad846dd0e4bbdfc6ee4b
```

## 🐛 常见问题

### Q1：注册后没有显示待激活界面？

**检查**：
- 是否有 JavaScript 错误（打开浏览器控制台）
- 后端是否正常运行（检查 Django 日志）
- 网络请求是否成功（查看 Network 标签）

### Q2：收不到激活邮件？

**解决**：
1. 检查垃圾邮件文件夹
2. 确认邮箱地址正确
3. 查看 Django 控制台日志
4. 验证邮件配置是否正确

### Q3：激活链接点击无效？

**原因**：
- 链接已过期（24小时）
- 链接已被使用
- 网络连接问题

**解决**：
- 重新注册账号
- 检查浏览器控制台
- 确认后端服务运行

### Q4：如何手动激活用户？

```bash
python manage.py shell

>>> from django.contrib.auth.models import User
>>> user = User.objects.get(email='test@example.com')
>>> user.is_active = True
>>> user.save()
>>> print(f"用户 {user.username} 已激活")
```
