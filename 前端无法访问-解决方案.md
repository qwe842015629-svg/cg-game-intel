## 🔍 常见问题诊断

### Q1: 浏览器显示"无法访问此网站"

**原因**: 服务器未启动

**解决**: 
```powershell
cd e:\小程序开发\游戏充值网站\frontend
npm run dev
```

**验证**: 看到 `VITE ready in XXX ms` 即成功

---

### Q2: 页面一直转圈加载

**原因1**: 后端服务器未启动

**解决**: 
```powershell
cd e:\小程序开发\游戏充值网站
python manage.py runserver
```

**原因2**: 端口被占用

**解决**: 
```powershell
# 查找占用进程
netstat -ano | findstr :5176

# 结束进程（替换 PID 为实际进程号）
taskkill /F /PID <进程号>
```

---

### Q3: 页面显示 404 或空白

**解决步骤**:

1. **检查前端是否运行**
   ```powershell
   netstat -ano | findstr :5176
   ```
   应该有输出，如果没有说明未启动

2. **检查后端是否运行**
   ```powershell
   netstat -ano | findstr :8000
   ```
   应该有输出，如果没有说明未启动

3. **重新安装依赖**
   ```powershell
   cd e:\小程序开发\游戏充值网站\frontend
   rm -rf node_modules
   npm install
   ```

---

### Q4: 浏览器显示端口拒绝连接

**原因**: IPv6 监听问题

**解决**: 使用 IPv4 地址
```
http://127.0.0.1:5176/
```

而不是
```
http://localhost:5176/
```

## 🆘 终极解决方案

如果以上都不行，执行完整重置：

```powershell

taskkill /F /IM node.exe
taskkill /F /IM python.exe

# 2. 删除依赖重新安装

Remove-Item node_modules -Recurse -Force
npm install

# 3. 重新启动
npm run dev
```
