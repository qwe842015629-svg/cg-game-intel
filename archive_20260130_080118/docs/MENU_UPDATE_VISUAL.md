## 🎯 关键特性

### 1. 视觉层次
```
层级1: 📰 全部资讯    ← 顶部，加粗，图标
       ═════════      ← 分割线
层级2: 游戏资讯        ← 下方，普通，有数量
       攻略教程
       充值指南
       活动公告
```

### 2. 颜色方案
```
默认状态:
  - 文字: #a8a8a8 (灰色)
  - 背景: rgba(0, 0, 0, 0.85) (深黑半透明)
  - 边框: #00ff88 (霓虹绿)

悬停状态:
  - 文字: #00ff88 (霓虹绿)
  - 背景: rgba(0, 255, 136, 0.1) (淡绿半透明)
```

### 3. 动画效果
```
菜单展开:
  opacity: 0 → 1
  transform: translateY(-10px) → translateY(0)
  duration: 0.3s

菜单收起:
  opacity: 1 → 0
  transform: translateY(0) → translateY(-10px)
  duration: 0.3s
```

## 📊 数据流向

```
前端组件 (Layout.vue)
    ↓
用户点击 "📰 全部资讯"
    ↓
Vue Router
    ↓
router.push('/articles')  ← 无 query 参数
    ↓
ArticlesPage.vue
    ↓
loadArticles()
    ↓
getArticles()  ← 无 params
    ↓
GET /api/articles/articles/
    ↓
Django Backend
    ↓
get_queryset()  ← category_name = None
    ↓
Article.objects.filter(status='published')
    ↓
返回所有已发布文章
    ↓
前端渲染
```
