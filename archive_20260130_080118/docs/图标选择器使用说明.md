## 🎯 使用方法

### 方法一：从图标库选择

1. **访问后台管理页面**
   ```
   http://127.0.0.1:8000/admin/main/homelayout/
   ```

2. **编辑任意板块**
   - 例如：点击"热门游戏板块"

3. **找到图标字段**
   - 显示为"🎨 板块图标"

4. **从图标库选择**
   - 点击分类标签（如："游戏"、"火热"等）
   - 浏览对应分类的图标
   - 点击任意图标即可选中

5. **实时预览**
   - 选中后，"当前图标"区域会实时更新
   - 显示选中的图标

6. **保存配置**
   - 点击页面底部的"保存"按钮

---

### 方法二：手动输入自定义图标

#### 输入 Emoji 表情

**支持的方式：**
- 直接复制粘贴 emoji：`🚀`
- 输入 emoji 代码（部分系统支持）

**示例：**
```
输入框中输入：🚀
结果：显示火箭图标
```

#### 输入图标类名（Font Awesome 等）

**格式：**
```
fas fa-gamepad
far fa-star
fab fa-github
```

**示例：**
```
输入框中输入：fas fa-fire
结果：前端渲染时会显示 Font Awesome 的火焰图标
```

#### 输入图片 URL

**格式：**
```
https://example.com/path/to/icon.png
/static/icons/custom-icon.svg
```

**示例：**
```
输入框中输入：https://cdn.example.com/game-icon.png
结果：前端会加载并显示该图片
```

## ⚙️ 技术实现

### 后端实现

**自定义 Widget：** `main/widgets.py`

```python
from django import forms

class IconPickerWidget(forms.Widget):
    """图标选择器小部件"""
    
    # 内置图标库
    ICON_LIBRARY = {
        '游戏': [...],
        '火热': [...],
        # ...
    }
    
    def render(self, name, value, attrs=None):
        # 渲染 HTML
        pass
```

### 表单应用

**各板块表单文件：**

```python
from ..widgets import IconPickerWidget

config_icon = forms.CharField(
    widget=IconPickerWidget(),  # 使用图标选择器
    help_text='从图标库选择或输入自定义图标'
)
```

### 前端集成

**HomePage.vue：**

```vue
<template>
  <span class="icon">
    <!-- 如果是 emoji，直接显示 -->
    <span v-if="isEmoji(icon)">{{ icon }}</span>
    
    <!-- 如果是图标类名，渲染 <i> 标签 -->
    <i v-else-if="isIconClass(icon)" :class="icon"></i>
    
    <!-- 如果是图片 URL，显示 <img> -->
    <img v-else :src="icon" alt="icon" />
  </span>
</template>
```

## 💡 最佳实践

### 1. 选择合适的图标
- **热门游戏：** 使用火焰 🔥、闪电 ⚡ 等表示热度
- **资讯内容：** 使用报纸 📰、喇叭 📢 等表示信息
- **安全功能：** 使用锁 🔒、盾牌 🛡️ 等表示安全
- **商业交易：** 使用钱袋 💰、信用卡 💳 等表示支付

### 2. 保持一致性
- 同一类型的板块使用相同风格的图标
- 避免混用 emoji 和 Font Awesome 图标

### 3. 考虑兼容性
- Emoji 在不同系统显示可能略有差异
- Font Awesome 图标需要确保前端已引入库
- 自定义图片需要注意 URL 可访问性

### 4. 性能优化
- 自定义图片建议使用 SVG 格式
- 图片大小控制在 50KB 以内
- 使用 CDN 加速图片加载

## ❓ 常见问题

### Q1：图标没有显示？

**可能原因：**
1. Emoji 不支持（旧系统）
2. Font Awesome 未引入
3. 图片 URL 无法访问

**解决方法：**
- 更换为常见 emoji
- 检查前端是否引入 Font Awesome
- 验证图片 URL 是否正确

### Q2：如何批量修改图标？

**方法：**
1. 进入 Django Shell：`python manage.py shell`
2. 批量更新：
```python
from main.models import HomeLayout
HomeLayout.objects.filter(section_key='hot_games').update(
    config={'icon': '🔥', 'title': '热门游戏', ...}
)
```

### Q3：能否上传图片文件？

**当前版本：** 支持输入图片 URL

**未来规划：** 可能支持直接上传图片文件

**临时方案：**
1. 将图片上传到 `/media/icons/` 目录
2. 在输入框中填写：`/media/icons/your-icon.png`

### Q4：支持动画图标吗？

**支持：** 可以使用 GIF 格式的图片 URL

**示例：**
```
https://example.com/animated-icon.gif
```
