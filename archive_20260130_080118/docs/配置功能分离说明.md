# 首页布局配置功能分离实现方案

## 🎯 问题分析

当前所有板块类型的配置字段都混在一起，导致：
- ❌ 编辑轮播图板块时，看到游戏数量、文章数量等无关字段
- ❌ 编辑游戏板块时，看到轮播间隔等无关配置
- ❌ 界面混乱，用户体验差

## ✅ 解决方案

### 方案一：为每个板块创建专属Admin类（推荐）

每个板块类型有自己独立的Admin配置类，显示专属字段。

#### 优势
- ✅ 完全分离，互不干扰
- ✅ 每个板块有定制化的配置界面
- ✅ 易于维护和扩展

#### 实现步骤

1. **创建专属表单**（已完成部分）
   - `BannerSectionForm` - 轮播图专用
   - `HotGamesSectionForm` - 热门游戏专用
   - `LatestNewsSectionForm` - 最新资讯专用
   - `FeaturesSectionForm` - 核心特性专用
   - `CategoriesSectionForm` - 游戏分类专用

2. **在Admin中动态选择表单**

```python
class HomeLayoutAdmin(admin.ModelAdmin):
    def get_form(self, request, obj=None, **kwargs):
        \"\"\"根据板块类型动态选择表单\"\"\"
        if obj:
            # 根据section_key选择对应的表单
            form_mapping = {
                'banner_section': BannerSectionForm,
                'hot_games': HotGamesSectionForm,
                'latest_news': LatestNewsSectionForm,
                'features': FeaturesSectionForm,
                'categories': CategoriesSectionForm,
            }
            kwargs['form'] = form_mapping.get(obj.section_key, BaseConfigForm)
        return super().get_form(request, obj, **kwargs)
```

3. **定制fieldsets**

```python
def get_fieldsets(self, request, obj=None):
    \"\"\"根据板块类型定制字段分组\"\"\"
    if obj and obj.section_key == 'banner_section':
        return (
            ('基本信息', {'fields': ('section_key', 'section_name')}),
            ('显示设置', {'fields': ('is_enabled', 'sort_order')}),
            ('轮播图配置', {
                'fields': (
                    'config_title',
                    'config_subtitle',
                    'config_auto_play',
                    'config_interval',
                    'config_show_indicators',
                    'config_show_arrows',
                )
            }),
        )
    elif obj and obj.section_key == 'hot_games':
        return (
            ('基本信息', {'fields': ('section_key', 'section_name')}),
            ('显示设置', {'fields': ('is_enabled', 'sort_order')}),
            ('热门游戏配置', {
                'fields': (
                    'config_title',
                    'config_subtitle',
                    'config_icon',
                    'config_display_count',
                    'config_show_more_button',
                    'config_layout',
                )
            }),
        )
    # ... 其他板块类型
```

### 方案三：分离为独立管理页面（最彻底）

为每个板块类型创建独立的管理入口。

#### 实现

创建代理模型：

```python
# models.py
class BannerSectionProxy(HomeLayout):
    class Meta:
        proxy = True
        verbose_name = '轮播图板块配置'

class HotGamesProxy(HomeLayout):
    class Meta:
        proxy = True
        verbose_name = '热门游戏板块配置'
```

注册独立Admin：

```python
# admin.py
@admin.register(BannerSectionProxy)
class BannerSectionAdmin(admin.ModelAdmin):
    form = BannerSectionForm
    # 只显示轮播图相关字段

@admin.register(HotGamesProxy)
class HotGamesAdmin(admin.ModelAdmin):
    form = HotGamesSectionForm
    # 只显示游戏相关字段
```
