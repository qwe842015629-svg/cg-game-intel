### 1. 查看前端页面

按 F12 打开控制台，应该看到：


### 2. 访问后台管理

打开浏览器访问：
```
http://127.0.0.1:8000/admin/
```

**登录后查看**：
- 左侧菜单中找到"页面底部管理"
- 点击展开，应该看到3个子菜单：
  - 底部板块
  - 底部链接
  - 底部配置

### 3. 测试编辑功能

#### 编辑板块标题
1. 进入"底部板块"管理
2. 点击任意一个板块（例如"关于我们"）
3. 修改标题为"关于我们 - 测试"
4. 保存

#### 验证前端同步
1. 回到前端页面：http://localhost:5176/
2. 刷新页面（Ctrl + F5）
3. 查看底部是否显示新标题"关于我们 - 测试"

#### 添加新链接
1. 进入"底部板块"管理
2. 点击"客服服务"板块
3. 在底部的"底部链接"内联编辑区域
4. 点击"添加另一个底部链接"
5. 填写：
   - 链接标题：测试链接
   - 链接地址：/test
   - 是否外部链接：不勾选
   - 是否启用：勾选
   - 排序：10
6. 保存

#### 验证新链接显示
1. 刷新前端页面
2. 查看"客服服务"板块是否显示新添加的"测试链接"

### 4. 测试配置功能

#### 修改版权信息
1. 进入"底部配置"管理
2. 点击唯一的配置记录
3. 修改版权文字为："© 2026 测试版权信息"
4. 保存

#### 验证版权显示
1. 刷新前端页面
2. 查看页面最底部是否显示新的版权信息

#### 隐藏版权信息
1. 返回"底部配置"管理
2. 取消勾选"显示版权信息"
3. 保存

#### 验证版权隐藏
1. 刷新前端页面
2. 版权信息应该不再显示

## 🔍 故障排查

### 如果前端不显示底部内容

#### 检查1：API是否正常
```bash
curl http://127.0.0.1:8000/api/footer/sections/
```
应该返回 JSON 数据，包含4个板块

#### 检查2：浏览器控制台是否有错误
按 F12 打开控制台，查看是否有红色错误信息

#### 检查3：网络请求是否成功
1. 按 F12 打开开发者工具
2. 切换到"网络"(Network)标签
3. 刷新页面
4. 查找 `footer/sections/` 请求
5. 查看状态码是否为 200

### 如果后台管理看不到菜单

#### 检查1：确认已登录
访问：http://127.0.0.1:8000/admin/
确保已登录管理员账号

#### 检查2：确认应用已注册
```bash
python manage.py shell
>>> from django.conf import settings
>>> 'footer' in settings.INSTALLED_APPS
True  # 应该返回 True
```

#### 检查3：清除浏览器缓存
按 Ctrl + Shift + Delete，清除缓存后重新访问
