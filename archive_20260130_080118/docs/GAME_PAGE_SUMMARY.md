## 🚀 如何使用

### 1. 后台管理
访问：http://127.0.0.1:8000/admin/game_page/

### 2. API 接口
基础地址：http://127.0.0.1:8000/api/game-pages/

主要接口：
- `GET /categories/` - 获取所有分类
- `GET /pages/` - 获取页面列表
- `GET /pages/{id}/` - 获取页面详情
- `GET /pages/hot_pages/` - 获取热门页面
- `POST /pages/{id}/like/` - 点赞页面

### 3. 前端调用
```typescript
import { getGamePages, getHotGamePages } from '@/api/gamePages'

// 获取所有页面
const pages = await getGamePages()

// 获取热门页面
const hotPages = await getHotGamePages()
```
