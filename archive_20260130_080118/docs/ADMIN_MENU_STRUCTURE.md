## 📊 菜单权重说明

SimpleUI的 `_weight` 参数控制菜单顺序：

```python
_weight: -1  →  首页管理（最前）
_weight:  1  →  游戏管理
_weight:  2  →  商品管理
_weight:  3  →  资讯管理
_weight:  4  →  用户管理
```

**规则**: 数字越小，越靠前

## 🎯 使用指南

### 第一次使用

1. **访问后台**
   ```
   URL: http://127.0.0.1:8000/admin/
   账号: admin
   密码: （您设置的密码）
   ```

2. **查看首页管理**
   - 点击左侧菜单 "⚙️ 首页管理"
   - 会展开两个子菜单

3. **管理首页布局**
   - 点击 "📐 首页布局"
   - 可以看到5个预设板块
   - 通过勾选框启用/禁用
   - 修改排序数字调整顺序

### 常用操作

#### 禁用某个首页板块

```
1. 进入 首页管理 → 首页布局
2. 点击要禁用的板块（如"最新资讯"）
3. 取消勾选 "是否启用"
4. 点击 "保存"
5. 前端立即生效（该板块不再显示）
```

#### 调整板块顺序

```
1. 进入 首页布局 列表页
2. 直接修改 "排序" 列的数字
   - 数字越小越靠前
   - 例如: 1, 2, 3, 4, 5
3. 点击页面底部的 "保存" 按钮
4. 刷新前端页面查看效果
```

#### 添加轮播图

```
1. 进入 首页管理 → 轮播图管理
2. 点击右上角 "增加轮播图"
3. 填写表单:
   - 标题: 例如 "原神新版本上线"
   - 描述: 简短描述
   - 徽章: 例如 "热卖中"
   - 上传图片
   - 配置按钮和链接
4. 保存
5. 前端轮播图自动更新
```

## 📱 响应式设计

SimpleUI支持响应式设计，在不同设备上都能良好显示：

```
💻 桌面端 (>1200px)
   - 侧边栏展开
   - 显示完整菜单名称和图标

📱 平板端 (768px - 1200px)
   - 侧边栏可折叠
   - 点击汉堡菜单展开

📱 手机端 (<768px)
   - 侧边栏默认收起
   - 点击图标展开菜单
```
