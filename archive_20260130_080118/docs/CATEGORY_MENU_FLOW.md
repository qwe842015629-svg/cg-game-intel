# 文章分类菜单动态加载流程图

## 🔄 完整数据流

```
┌─────────────────────────────────────────────────────────────┐
│                     管理员操作                                │
│  http://127.0.0.1:8000/admin/game_article/articlecategory/  │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 1. 添加/编辑分类
                        │    - 设置名称、排序、启用状态
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│                   MySQL 数据库                               │
│  表: article_category                                        │
├─────────────────────────────────────────────────────────────┤
│  id │ name     │ sort_order │ is_active │ articles_count   │
│  1  │ 游戏资讯  │     1      │   True    │      5          │
│  2  │ 攻略教程  │     2      │   True    │      3          │
│  3  │ 充值指南  │     3      │   True    │      2          │
│  4  │ 活动公告  │     4      │   True    │      8          │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 2. 前端页面加载
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│              前端 Layout.vue (onMounted)                     │
│  import { getArticleCategories } from '../api/articles'     │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 3. 调用API
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│              API请求                                         │
│  GET http://127.0.0.1:8000/api/articles/categories/         │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 4. Django处理
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│           ArticleCategoryViewSet                            │
│  queryset = ArticleCategory.objects.filter(is_active=True)  │
│  ordering = ['sort_order']                                  │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 5. 序列化
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│           ArticleCategorySerializer                          │
│  计算 articles_count (该分类下的文章数)                       │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 6. 返回JSON
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│              JSON响应                                        │
│  [                                                           │
│    {                                                         │
│      "id": 1,                                                │
│      "name": "游戏资讯",                                      │
│      "sort_order": 1,                                        │
│      "is_active": true,                                      │
│      "articles_count": 5                                     │
│    },                                                        │
│    ...                                                       │
│  ]                                                           │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 7. 前端接收
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│              Vue响应式数据                                    │
│  articleCategories.value = [...]                            │
└───────────────────────┬─────────────────────────────────────┘
                        │
                        │ 8. 模板渲染
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│              前端页面显示                                     │
│  ┌──────────────┐                                           │
│  │  资讯 ▼      │  ← 用户鼠标悬停                            │
│  └──────┬───────┘                                           │
│         ↓                                                    │
│    ┌───────────────────┐                                    │
│    │ 游戏资讯 (5)      │  ← v-for 动态渲染                   │
│    │ 攻略教程 (3)      │                                     │
│    │ 充值指南 (2)      │                                     │
│    │ 活动公告 (8)      │                                     │
│    └───────────────────┘                                    │
└─────────────────────────────────────────────────────────────┘
```

## 🎯 关键特性

### 1. 完全数据驱动
```
后台修改 → 数据库更新 → API获取 → 前端渲染
```

### 2. 自动排序
```
sort_order: 1, 2, 3, 4  →  菜单顺序: 1, 2, 3, 4
```

### 3. 动态启用/禁用
```
is_active: True  → 显示在菜单
is_active: False → 隐藏
```

### 4. 文章数量统计
```
每个分类旁显示: (文章数)
例如: 游戏资讯 (5)
```

## 📊 组件交互

```
Layout.vue
├─ onMounted()
│  └─ loadArticleCategories()
│     ├─ API调用: getArticleCategories()
│     └─ 更新: articleCategories.value
│
└─ Template
   └─ v-for="category in articleCategories"
      └─ RouterLink :to="'/articles?category=' + category.name"
```

## 🔐 安全性

- ✅ 后端只返回 `is_active=True` 的分类
- ✅ 只读API（ReadOnlyModelViewSet）
- ✅ 前端容错处理（API失败时使用默认值）

## 📱 响应式更新

```
管理员添加新分类
    ↓
用户刷新页面
    ↓
自动显示新分类
```
