# 🎯 板块专属配置使用说明

## ✅ 已完成功能分离

现在每个板块类型都有独立的配置表单，编辑时只会看到相关的配置字段！

## 📋 五大板块专属表单

### 1️⃣ 轮播图板块 (`banner_section`)

**专属配置字段：**
- ✅ 自动轮播：是否自动切换
- ⏱️ 轮播间隔：切换时间（毫秒）
- ⚫ 显示指示器：底部圆点
- ◀️ 显示箭头：左右切换按钮
- ⏸️ 鼠标悬停暂停：是否暂停自动轮播

**适用场景：**首页顶部大图轮播，展示活动、新游戏等

---

### 2️⃣ 热门游戏板块 (`hot_games`)

**专属配置字段：**
- 🎮 显示游戏数量：4-20个可选
- 📐 布局样式：网格/列表/轮播
- ⭐ 显示游戏评分：星级评分开关
- 💰 显示折扣信息：折扣标签开关
- ➕ 显示"查看更多"按钮

**适用场景：**展示热门游戏充值入口

---

### 3️⃣ 最新资讯板块 (`latest_news`)

**专属配置字段：**
- 📄 显示资讯数量：3-12条可选
- 🏷️ 显示资讯分类：分类标签开关
- 👤 显示作者信息：作者名称开关
- 📅 显示发布日期：时间显示开关
- 🖼️ 显示缩略图：封面图开关
- 📐 布局样式：卡片/列表/时间线

**适用场景：**展示游戏公告、活动资讯、攻略等

---

### 4️⃣ 核心特性板块 (`features`)

**专属配置字段：**
- 特性1：图标 + 标题 + 描述（例如：⚡ 极速到账）
- 特性2：图标 + 标题 + 描述（例如：🔒 安全可靠）
- 特性3：图标 + 标题 + 描述（例如：💰 超值优惠）
- 📐 布局方式：横向/纵向/网格
- ✨ 开启动画效果：滚动动画开关

**适用场景：**展示平台优势、服务特点

---

### 5️⃣ 游戏分类板块 (`categories`)

**专属配置字段：**
- 📂 显示"全部分类"选项
- 📐 布局样式：卡片/图标/列表/网格
- 📊 每行显示数量：3-6个
- 🔢 显示游戏数量：每个分类下的游戏数
- 🎨 显示分类图标
- ✨ 启用悬停效果：鼠标hover特效
- 🔥 显示热门标签：HOT标签
- 📋 显示分类数量：6-12个

**适用场景：**游戏分类导航，快速找到目标游戏

---

## 🎨 使用方法

### 第一步：访问后台管理
```
http://127.0.0.1:8000/admin/main/homelayout/
```

### 第二步：选择要编辑的板块
- 点击任意板块名称进入编辑页面

### 第三步：根据板块类型配置
编辑不同板块时，表单会**自动显示对应的专属字段**：

- 编辑"轮播图板块" → 显示轮播专属配置
- 编辑"热门游戏板块" → 显示游戏列表专属配置
- 编辑"最新资讯板块" → 显示资讯专属配置
- 编辑"核心特性板块" → 显示特性配置
- 编辑"游戏分类板块" → 显示分类专属配置

### 第四步：保存配置
- 点击"保存"或"保存并继续编辑"
- 配置自动转换为JSON格式存入数据库

## 🔍 技术实现

### 表单文件结构
```
main/forms/
├── __init__.py                 # 导出所有表单
├── base_form.py                # 基础通用表单
├── banner_form.py              # 轮播图专属表单
├── hot_games_form.py           # 热门游戏专属表单
├── latest_news_form.py         # 最新资讯专属表单
├── features_form.py            # 核心特性专属表单
└── categories_form.py          # 游戏分类专属表单
```

### 动态表单选择机制
在 `main/admin.py` 的 `HomeLayoutAdmin` 中：

```python
def get_form(self, request, obj=None, **kwargs):
    """根据板块类型动态选择表单"""
    FORM_MAPPING = {
        'banner_section': BannerSectionForm,
        'hot_games': HotGamesSectionForm,
        'latest_news': LatestNewsSectionForm,
        'features': FeaturesSectionForm,
        'categories': CategoriesSectionForm,
    }
    form_class = FORM_MAPPING.get(obj.section_key, BaseConfigForm)
    if form_class:
        kwargs['form'] = form_class
    return super().get_form(request, obj, **kwargs)
```

## ✨ 优势对比

### ❌ 之前（配置混乱）
- 编辑任何板块都看到所有配置字段
- 不知道哪些字段是当前板块需要的
- 容易误配置无关字段
- 表单字段过多，眼花缭乱

### ✅ 现在（配置分离）
- 编辑轮播图只看到轮播相关配置
- 编辑热门游戏只看到游戏列表配置
- 每个板块配置清晰明了
- 避免误操作，提升编辑体验

## 🎯 常见配置场景

### 场景1：调整轮播速度
1. 进入"轮播图板块"编辑
2. 修改"轮播间隔"字段（如改为3000毫秒=3秒）
3. 保存

### 场景2：增加热门游戏展示数量
1. 进入"热门游戏板块"编辑
2. 修改"显示游戏数量"（如改为12）
3. 保存

### 场景3：修改平台优势内容
1. 进入"核心特性板块"编辑
2. 分别编辑特性1/2/3的图标、标题、描述
3. 保存

### 场景4：调整资讯显示方式
1. 进入"最新资讯板块"编辑
2. 切换"布局样式"（卡片→列表）
3. 勾选/取消需要显示的元素（分类、日期等）
4. 保存

## 📝 注意事项

1. **基础配置**：所有板块都支持标题、副标题配置
2. **JSON预览**：保存后可在"当前配置预览"查看生成的JSON
3. **排序控制**：通过`sort_order`字段控制板块显示顺序
4. **启用/禁用**：通过`is_enabled`开关控制板块是否在首页显示

## 🚀 扩展指南

如需添加新的配置字段，只需：
1. 在对应的表单文件中添加字段定义
2. 在`__init__()`中添加初始值读取
3. 在`clean()`中添加JSON组装逻辑

示例：为轮播图添加"切换动画"配置
```python
# 在 banner_form.py 中添加
config_transition = forms.ChoiceField(
    choices=[
        ('fade', '淡入淡出'),
        ('slide', '滑动'),
        ('zoom', '缩放'),
    ],
    initial='fade',
    label='切换动画'
)

# 在 clean() 中添加
config['transition'] = cleaned_data.get('config_transition', 'fade')
```
