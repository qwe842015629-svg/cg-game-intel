# 游戏页面管理 - 快速使用指南

## 🎯 快速开始

### 1. 访问后台管理

打开浏览器访问：
```
http://127.0.0.1:8000/admin/game_page/
```

登录后可以看到：
- **页面分类** - 管理游戏页面的分类
- **游戏页面** - 创建和管理游戏相关内容页面

---

## 📝 创建第一个游戏页面

### 步骤 1: 创建分类（如果需要）

1. 点击 **"页面分类"**
2. 点击右上角 **"增加游戏页面分类"**
3. 填写信息：
   - **分类名称**: 例如 "游戏攻略"
   - **分类描述**: 例如 "各类游戏的攻略指南"
   - **排序**: 数字越小越靠前
   - **是否启用**: 勾选
4. 点击 **"保存"**

### 步骤 2: 创建游戏页面

1. 点击 **"游戏页面"**
2. 点击右上角 **"增加游戏页面"**
3. 填写基本信息：
   - **页面标题**: 例如 "王者荣耀新手攻略"
   - **URL别名**: 自动生成或手动输入（如 wzry-guide）
   - **页面分类**: 选择刚才创建的分类
   - **关联游戏**: 可选择关联的游戏
   - **作者名称**: 默认"游戏小编"
   - **封面图**: 点击上传图片

4. 填写内容：
   - **摘要**: 简短介绍（最多500字）
   - **页面内容**: 富文本编辑器，支持HTML

5. 设置状态：
   - **状态**: 选择"已发布"
   - **是否置顶**: 重要内容可勾选
   - **是否热门**: 热门内容可勾选
   - **是否推荐**: 推荐内容可勾选
   - **发布时间**: 选择发布时间

6. 点击 **"保存"**

## 🔍 查看和管理页面

### 筛选功能

在游戏页面列表，可以通过右侧筛选器快速查找：
- **按状态筛选**: 草稿/已发布/已归档
- **按分类筛选**: 选择特定分类
- **按游戏筛选**: 选择特定游戏
- **按标记筛选**: 置顶/热门/推荐

### 搜索功能

在右上角搜索框中输入关键词，可搜索：
- 页面标题
- 摘要内容
- 正文内容
- 作者名称

### 排序功能

点击列表表头可以按以下字段排序：
- 标题
- 分类
- 浏览次数
- 点赞数
- 发布时间
- 创建时间

## 📊 查看统计数据

每个页面都会自动统计：
- **浏览次数**: 前端用户访问时自动+1
- **点赞数**: 用户点赞时自动+1

这些数据在列表和详情页都可以看到。

## 🌐 前端访问

### API接口地址

```
# 获取所有分类
GET http://127.0.0.1:8000/api/game-pages/categories/

# 获取所有页面
GET http://127.0.0.1:8000/api/game-pages/pages/

# 获取热门页面
GET http://127.0.0.1:8000/api/game-pages/pages/hot_pages/

# 获取页面详情
GET http://127.0.0.1:8000/api/game-pages/pages/{id}/

# 根据slug获取详情
GET http://127.0.0.1:8000/api/game-pages/pages/by_slug/?slug=wzry-guide
```

### 使用示例（TypeScript）

```typescript
import { 
  getGamePages, 
  getHotGamePages, 
  getGamePageDetail 
} from '@/api/gamePages'

// 获取所有页面
const pages = await getGamePages()

// 获取特定分类的页面
const categoryPages = await getGamePages({ 
  category: 1 
})

// 获取热门页面
const hotPages = await getHotGamePages()

// 获取页面详情
const page = await getGamePageDetail(1)

// 搜索页面
const searchResults = await getGamePages({ 
  search: '攻略' 
})
```

## 💡 最佳实践

### 1. 内容规范
- **标题**: 简洁明了，不超过50字
- **摘要**: 概括核心内容，100-200字
- **正文**: 结构清晰，使用标题分段
- **图片**: 压缩后上传，建议500KB以内
- **slug**: 使用英文或拼音，便于SEO

### 2. 分类规划
建议的分类：
- 游戏攻略
- 游戏资讯
- 新手指南
- 活动公告
- 玩家心得
- 常见问题

### 3. 标记使用
- **置顶**: 最多设置3-5个重要内容
- **热门**: 根据浏览量自动设置
- **推荐**: 优质内容手动标记

### 4. 发布流程
1. 创建草稿
2. 填写完整内容
3. 预览检查
4. 设置发布时间
5. 修改状态为"已发布"

## 🛠️ 常见问题

### Q1: 如何上传图片到内容中？

**方法1**: 先在Django Admin的其他地方上传图片，然后复制URL
**方法2**: 使用外部图床，获取图片URL后插入

```html
<img src="图片URL" alt="描述">
```

### Q2: 如何设置置顶？

在编辑页面的"状态设置"部分，勾选"是否置顶"即可。

### Q3: 草稿会在前端显示吗？

不会。只有状态为"已发布"且发布时间早于当前时间的页面才会在前端显示。

### Q4: 如何修改URL别名？

在"基本信息"部分的"URL别名"字段中修改，注意要保证唯一性。

### Q5: 浏览次数如何统计？

前端用户访问页面详情时自动+1，管理员访问不计数。

### Q6: 如何批量管理页面？

在列表页勾选多个页面，然后在"动作"下拉框选择操作（如批量删除）。
