## 🚀 使用方法

### 管理首页布局

1. **登录后台**
   ```
   访问: http://127.0.0.1:8000/admin/
   用户名: admin
   ```

2. **进入首页管理**
   - 点击左侧菜单 "⚙️ 首页管理"
   - 点击 "📐 首页布局"

3. **启用/禁用板块**
   - 点击要编辑的板块
   - 勾选/取消 "是否启用"
   - 点击保存

4. **调整板块顺序**
   - 在列表页直接修改"排序"列的数字
   - 点击页面底部的"保存"按钮
   - 数字越小，板块越靠前

5. **配置板块参数**
   - 进入板块编辑页面
   - 在"板块配置"字段输入JSON
   - 例如: `{"display_count": 8, "show_badge": true}`

### 管理轮播图

1. **进入轮播图管理**
   - 点击 "⚙️ 首页管理" → "🖼️ 轮播图管理"

2. **添加轮播图**
   - 点击右上角"增加轮播图"
   - 填写标题、描述、徽章
   - 上传图片（推荐尺寸：1920x600px）
   - 配置按钮和链接
   - 设置排序和状态
   - 保存
