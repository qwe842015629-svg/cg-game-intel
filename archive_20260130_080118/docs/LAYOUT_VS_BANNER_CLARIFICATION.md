# 首页布局 vs 轮播图管理 - 功能区分说明

## 🎯 问题描述

原有设计中，**首页布局**和**轮播图管理**功能存在重复和混淆：

### ❌ 原有问题

1. **首页布局**中有"轮播图区域"（hero_carousel）选项
2. **轮播图管理**也在管理轮播图内容
3. 两者功能边界不清晰，容易混淆

```
首页布局                轮播图管理
    ↓                      ↓
轮播图区域 ←→ 重复 ←→ 轮播图内容
```

## ✅ 修复方案

### 清晰的职责划分

```

│  ✓ 核心特性板块    [启用/禁用]  排序: 2        │
│  ✓ 热门游戏板块    [启用/禁用]  排序: 3        │
```

## 📊 功能对比表

| 功能 | 首页布局管理 | 轮播图管理 |
|------|-------------|-----------|
| **管理对象** | 首页板块（section） | 轮播图内容（banner） |
| **控制范围** | 整个板块的显示/隐藏 | 单个轮播图的内容 |
| **主要功能** | 板块开关、排序 | 图片、文字、链接管理 |
| **数据表** | `layout` | `banner` |
| **访问路径** | `/admin/main/homelayout/` | `/admin/main/banner/` |
| **菜单位置** | 首页管理 > 首页布局 | 首页管理 > 轮播图管理 |

## 🔧 修改内容

### 1. 模型字段修改 (main/models.py)

**修改前**:
```python
SECTION_CHOICES = [
    ('hero_carousel', '轮播图区域'),  # ❌ 容易混淆
    ('features', '核心特性'),
    ('hot_games', '热门游戏'),
    ('categories', '游戏分类'),
    ('latest_news', '最新资讯'),
]
```

**修改后**:
```python
SECTION_CHOICES = [
    ('banner_section', '轮播图板块'),  # ✅ 明确是"板块"
    ('features', '核心特性板块'),
    ('hot_games', '热门游戏板块'),
    ('categories', '游戏分类板块'),
    ('latest_news', '最新资讯板块'),
]
```

**关键变化**:
- `hero_carousel` → `banner_section` (更明确的命名)
- 所有选项统一加"板块"后缀，强调是板块级别的控制

## 📝 使用场景说明

### 场景1: 关闭整个轮播图板块

**在首页布局中**:
```
轮播图板块  [禁用] ← 点击这里
```

**效果**: 
- 整个轮播图区域在首页不显示
- 即使有10个启用的轮播图，也不会显示

### 场景2: 管理轮播图内容

**在轮播图管理中**:
```
📸 轮播图1: 王者荣耀  [启用]
📸 轮播图2: 原神      [禁用] ← 这个不显示
📸 轮播图3: 崩坏3     [启用]
```

**效果**:
- 只有启用的轮播图会在轮播中显示
- 前提是"轮播图板块"在首页布局中是启用的

### 场景3: 调整首页板块顺序

**在首页布局中**:
```
排序 1: 轮播图板块
排序 2: 热门游戏板块
排序 3: 核心特性板块  ← 调整这个到最前面
```

**效果**:
- 首页板块按新的顺序显示
- 不影响各板块内部的内容

### 数据更新脚本

```python
from main.models import HomeLayout

# 更新现有的 hero_carousel 记录
try:
    old_layout = HomeLayout.objects.get(section_key='hero_carousel')
    old_layout.section_key = 'banner_section'
    old_layout.section_name = '轮播图板块'
    old_layout.save()
    print("✅ 已更新轮播图板块配置")
except HomeLayout.DoesNotExist:
    print("ℹ️ 未找到旧的轮播图配置")
```

## 🔍 常见问题

### Q1: 为什么我启用了轮播图，但首页不显示？

**A**: 检查两个地方：
1. **首页布局** > 轮播图板块是否启用
2. **轮播图管理** > 是否有启用的轮播图

### Q2: 首页布局和轮播图管理有什么区别？

**A**: 
- **首页布局**: 控制"要不要显示轮播图板块" + 板块顺序
- **轮播图管理**: 控制"显示哪些轮播图" + 轮播图内容

### Q3: 我可以只禁用某个轮播图吗？

**A**: 可以！在**轮播图管理**中禁用特定轮播图即可。

### Q4: 如何调整轮播图的显示顺序？

**A**: 在**轮播图管理**中调整各轮播图的排序数字。

### Q5: 迁移后会丢失数据吗？

**A**: 不会。迁移只是更新字段名称，数据会保留。
