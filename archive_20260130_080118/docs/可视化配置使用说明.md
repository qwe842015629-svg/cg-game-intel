# 首页布局可视化配置使用说明

## ✅ 已完成功能

您现在可以通过**可视化表单**来配置首页板块内容，无需手动编写JSON代码！

## 🎯 使用方法

### 1. 访问后台管理
打开浏览器访问：http://127.0.0.1:8000/admin/main/homelayout/

### 2. 编辑板块配置
点击任一板块进入编辑页面，您将看到以下可视化配置项：

#### 📝 板块内容配置（可视化编辑）
- **板块标题**：显示在前端的板块标题
  - 例如：`热门游戏`、`核心特性`
  
- **板块副标题**：显示在标题下方的描述文字
  - 例如：`最受欢迎的游戏充值`
  
- **图标**：可以使用Emoji或图标代码
  - 例如：`🔥`、`⚡`、`✨`
  
- **显示数量**：该板块显示的项目数量（1-50）
  - 用于控制显示多少个游戏、文章等
  - 例如：`8` 表示显示8个项目
  
- **显示"查看更多"按钮**：勾选后在底部显示按钮

#### 🎪 轮播图板块专用配置
仅在编辑"轮播图板块"时需要配置：

- **自动轮播**：是否自动切换轮播图
- **轮播间隔（毫秒）**：自动切换的时间间隔
  - 1000毫秒 = 1秒
  - 建议设置为 3000-8000

## 📊 配置示例

### 示例1：配置热门游戏板块

1. 点击"热门游戏板块"进入编辑
2. 填写表单：
   - 板块标题：`超人气游戏`
   - 板块副标题：`玩家最爱的充值选择`
   - 图标：`🔥`
   - 显示数量：`8`
   - 勾选"显示查看更多按钮"
3. 点击保存

**自动生成的JSON配置**：
```json
{
  "title": "超人气游戏",
  "subtitle": "玩家最爱的充值选择",
  "icon": "🔥",
  "display_count": 8,
  "show_more_button": true
}
```

### 示例2：配置轮播图板块

1. 点击"轮播图板块"进入编辑
2. 填写表单：
   - 板块标题：`精彩活动`
   - 板块副标题：`最新优惠与福利`
   - 勾选"自动轮播"
   - 轮播间隔：`5000` （5秒）
3. 点击保存

**自动生成的JSON配置**：
```json
{
  "title": "精彩活动",
  "subtitle": "最新优惠与福利",
  "auto_play": true,
  "interval": 5000,
  "show_more_button": true
}
```

### 示例3：配置最新资讯板块

1. 点击"最新资讯板块"进入编辑
2. 填写表单：
   - 板块标题：`游戏资讯`
   - 板块副标题：`热门攻略 · 最新动态`
   - 图标：`📰`
   - 显示数量：`6`
3. 点击保存

## 🔍 查看配置预览

在编辑页面底部，展开"📊 当前配置预览"，可以看到自动生成的JSON配置。

## 💡 优势特点

### ✨ 之前（需要手动写JSON）
```json
{
  "title": "热门游戏",
  "subtitle": "最受欢迎的游戏充值",
  "display_count": 8,
  "show_more_button": true
}
```
- ❌ 容易写错格式
- ❌ 需要记住字段名
- ❌ 引号、逗号容易出错

### ✨ 现在（可视化表单）
- ✅ 填写表单即可
- ✅ 有输入提示和说明
- ✅ 自动验证数据
- ✅ 自动生成正确的JSON
- ✅ 支持预览

## 🚀 技术实现

系统使用自定义的Django表单（`HomeLayoutAdminForm`）：
1. 将JSON配置拆分成独立的表单字段
2. 保存时自动组装成JSON格式
3. 编辑时自动从JSON中读取数据填充表单

## 📋 字段对照表

| 表单字段 | JSON字段名 | 说明 |
|---------|-----------|------|
| 板块标题 | `title` | 前端显示的标题 |
| 板块副标题 | `subtitle` | 标题下方的描述 |
| 图标 | `icon` | Emoji或图标代码 |
| 显示数量 | `display_count` | 显示项目数量 |
| 显示"查看更多"按钮 | `show_more_button` | 是否显示按钮 |
| 自动轮播 | `auto_play` | 轮播图专用 |
| 轮播间隔 | `interval` | 轮播图专用（毫秒） |

## ⚠️ 注意事项

1. **必填字段**：板块标识和板块名称是必填的
2. **数值范围**：显示数量建议在1-50之间
3. **轮播间隔**：建议设置在1000-10000毫秒之间
4. **保存后刷新**：修改配置后需要刷新前端页面才能看到效果

## 🎨 前端效果

配置保存后，前端页面将自动：
1. 使用您设置的标题和副标题
2. 按照显示数量展示内容
3. 根据配置显示或隐藏"查看更多"按钮
4. 轮播图按照设置的间隔自动切换

## 📞 如需帮助

如果在使用过程中遇到问题，可以：

2. 检查Django后台是否有报错信息
3. 检查浏览器控制台是否有错误
