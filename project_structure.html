🎮 游戏充值网站 - 项目架构与路径清单

📂 项目根目录结构

根路径: e:\小程序开发\游戏充值网站\

目录/文件 说明
frontend/ 前端 Vue 3 + Vite 项目根目录

manage.py Django 项目管理脚本 (入口)
requirements.txt 后端 Python 依赖包列表
PROJECT_STRUCTURE.md 项目结构说明文档
MAIN_DOCUMENTATION.md 项目主技术文档
venv/ Python 虚拟环境目录

🏗️ 后端架构 (Django)

主要应用模块位于项目根目录下：

路径 模块名称 功能描述
main/ 首页管理 轮播图管理、首页布局配置、通用配置
game_product/ 游戏产品 游戏分类、游戏列表、商品类型管理
game_article/ 资讯文章 文章发布、分类管理、评论系统
game_page/ 游戏页面 单页游戏介绍、富文本内容、页面分类
game_product_show/ 产品展示 独立的产品展示页面管理
customer_service/ 客服系统 联系方式配置、常见问题 (FAQ) 管理
footer/ 底部管理 网站底部链接、板块配置
users/ 用户系统 用户注册、登录、个人信息管理
orders/ 订单系统 充值订单处理与记录
payments/ 支付系统 支付接口对接与回调处理

关键配置文件

  • game_recharge/settings.py: 全局配置文件 (数据库、跨域、应用注册)
  • game_recharge/urls.py: 全局路由入口
  • scripts/: 包含数据初始化、翻译、清理等自动化维护脚本

🎨 前端架构 (Vue 3)

路径: frontend/src/

路径 类型 功能描述
api/ API 接口 封装与后端交互的请求 (如 games.ts, auth.ts)
views/HomePage.vue 页面 首页 (轮播图 + 热门游戏展示)
views/GamesPage.vue 页面 游戏列表 (搜索、筛选、游戏网格)
views/GameDetailPage.vue 页面 游戏详情 (充值表单、区服选择、支付方式)
views/ProfilePage.vue 页面 个人中心 (用户信息、订单记录)
components/ui/ UI 组件 基础组件：Button, Input, Dialog 等
components/Layout.vue 布局组件 包含通用的 Header 导航栏和 Footer
stores/auth.ts 状态管理 用户登录状态、信息管理 (Pinia)
router/index.ts 路由配置 页面跳转规则定义
assets/ 静态资源 全局样式 (main.css)、图片等

🛠️ 关键维护脚本

脚本/命令 用途
python manage.py runserver 启动 启动后端 Django 开发服务器
npm run dev 启动 启动前端 Vite 开发服务器 (在 frontend 目录运行)
cleanup.bat 维护 Windows 环境清理脚本
scripts/init_*.py 数据 各模块数据初始化脚本集合
