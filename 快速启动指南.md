# 🚀 快速启动指南

## ❓ 遇到"无法访问此网站"错误？

如果浏览器显示 `ERR_CONNECTION_REFUSED`，说明服务器没有启动。

---

## ⚡ 最简单的启动方法（推荐）

### 方法1：使用一键启动脚本

1. **双击 `启动后台.bat`** 文件
   - 会自动启动 Django 后台服务器
   - 窗口保持打开，服务器就在运行

2. **双击 `启动前端.bat`** 文件（如需访问前端）
   - 会自动启动 Vue 前端服务器
   - 窗口保持打开，服务器就在运行

3. **访问网站**
   - 后台管理：http://127.0.0.1:8000/admin/
   - 前端页面：http://localhost:5176/

---

## 📝 手动启动方法

### 启动后台服务器

**在 PowerShell 中执行**：

```powershell
# 1. 进入项目目录
cd e:\小程序开发\游戏充值网站

# 2. 激活虚拟环境
.\venv\Scripts\activate

# 3. 启动服务器
python manage.py runserver
```

**成功标志**：
```
Starting development server at http://127.0.0.1:8000/
```

### 启动前端服务器（可选）

**在新的 PowerShell 窗口执行**：

```powershell
# 1. 进入前端目录
cd e:\小程序开发\游戏充值网站\frontend

# 2. 启动前端
npm run dev
```

**成功标志**：
```
VITE ready in XXX ms
➜  Local:   http://localhost:5176/
```

---

## 🔍 常见问题解答

### Q1: 双击 bat 文件后窗口一闪就消失了
**A**: 右键点击 bat 文件 → 选择"以管理员身份运行"

### Q2: 提示"找不到 python"
**A**: 需要先安装 Python 3.10+，并添加到系统 PATH

### Q3: 提示"找不到 venv"
**A**: 在项目目录运行：
```powershell
python -m venv venv
```

### Q4: 端口被占用
**A**: 使用不同端口：
```powershell
python manage.py runserver 8001
```

### Q5: 前端启动失败
**A**: 重新安装依赖：
```powershell
cd frontend
npm install
```

---

## 🎯 访问地址

### 后台管理系统
```
http://127.0.0.1:8000/admin/
```
**用途**：管理游戏、文章、用户等后台功能

### 前端页面
```
http://localhost:5176/
```
**用途**：用户访问的网站前端

### API 接口
```
http://127.0.0.1:8000/api/
```
**用途**：前后端数据交互

---

## 📋 启动检查清单

启动前确认以下事项：

- [ ] Python 已安装（3.10+）
- [ ] Node.js 已安装（18+）
- [ ] MySQL 数据库已启动
- [ ] 项目依赖已安装
- [ ] 数据库已迁移

---

## 🆘 需要帮助？

如果启动失败，请：

1. **查看错误信息**：启动窗口中的红色文字
2. **检查端口占用**：确保 8000 和 5176 端口未被占用
3. **查看详细文档**：`USER_GUIDE.md`

---

**提示**：第一次启动可能需要几秒钟，请耐心等待！

**记住**：启动后窗口不要关闭，关闭窗口 = 停止服务器
