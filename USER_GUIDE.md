# 🎮 游戏充值网站 - 小白用户使用指南

## 🎯 欢迎使用游戏充值网站！

这份指南将帮助您快速上手使用我们的游戏充值网站后台管理系统。无需编程知识，跟着步骤操作即可！

---

## 🚀 第一步：启动系统

### 1.1 启动后台服务器

**如果您是开发者**：
1. 打开命令行窗口（PowerShell 或 CMD）
2. 进入项目目录：`cd e:\小程序开发\游戏充值网站`
3. 激活虚拟环境：`.\venv\Scripts\activate`
4. 启动服务器：`python manage.py runserver`

**如果您不是开发者**：
请让技术人员帮您启动服务器，您只需要知道访问地址即可。

### 1.2 访问后台管理

打开浏览器，输入地址：
```
http://127.0.0.1:8000/admin/
```

### 1.3 登录系统

使用管理员账号登录：
- **用户名**: admin
- **密码**: 您设置的密码

## 📝 第二步：管理游戏页面

### 2.1 添加游戏页面

1. **找到菜单**：在左侧菜单中找到"游戏页面管理"
2. **点击子菜单**：点击"游戏页面"
3. **添加新页面**：点击右上角的"增加游戏页面"

**填写页面信息**：
- **页面标题**：比如"王者荣耀新手攻略"
- **URL别名**：系统自动生成，也可以手动设置
- **页面分类**：选择一个分类（如果没有，先去"页面分类"创建）
- **关联游戏**：可选择相关的游戏
- **作者名称**：默认"游戏小编"，可以修改
- **封面图**：点击"选择文件"上传图片
- **摘要**：简短描述，100字左右
- **页面内容**：使用富文本编辑器编写内容

**设置发布选项**：
- **状态**：选择"已发布"
- **是否置顶**：重要内容可以勾选
- **是否热门**：热门内容可以勾选
- **是否推荐**：推荐内容可以勾选

点击"保存"完成创建！

### 2.2 管理页面分类

1. **找到菜单**：左侧菜单"游戏页面管理"
2. **点击子菜单**：点击"页面分类"
3. **添加分类**：点击右上角"增加游戏页面分类"

**填写分类信息**：
- **分类名称**：比如"游戏攻略"、"新手指南"
- **分类描述**：对该分类的说明
- **排序**：数字越小越靠前
- **是否启用**：勾选表示启用该分类

### 2.3 编辑现有页面

1. 在"游戏页面"列表中找到要编辑的页面
2. 点击页面标题或"更改"链接
3. 修改需要的内容
4. 点击"保存"

## 💡 第七步：实用技巧

### 7.1 快速搜索
- 在任何列表页面，使用右上角的搜索框可以快速找到内容
- 支持按标题、名称、内容等搜索

### 7.2 批量操作
- 在列表页面可以勾选多个项目进行批量操作
- 比如批量删除、批量修改状态等

### 7.3 筛选功能
- 使用右侧的筛选器可以按条件筛选内容
- 比如按状态、分类、时间等筛选

### 7.4 富文本编辑器使用

**基本格式**：
- 标题：使用 `## 一级标题` 或 `### 二级标题`
- 加粗：使用 `**加粗文字**`
- 斜体：使用 `*斜体文字*`
- 列表：使用 `-` 或 `1.` 开头

**插入图片**：
1. 先上传图片到服务器
2. 复制图片URL
3. 使用格式：`![描述](图片URL)`

## 🔧 第八步：常见问题解答

### Q1：添加的内容在前台不显示？
**A**：检查状态是否设置为"已发布"，发布时间是否正确。

### Q2：上传图片失败？
**A**：检查图片格式（建议JPG、PNG），文件大小（建议小于5MB）。

### Q3：菜单不显示新功能？
**A**：请让技术人员重启服务器，然后按 `Ctrl+Shift+R` 强制刷新浏览器。

### Q4：忘记密码怎么办？
**A**：联系系统管理员重置密码。

### Q5：如何备份数据？
**A**：请让技术人员使用 `python manage.py dumpdata` 命令备份。
